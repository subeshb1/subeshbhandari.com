# Using the Callout component

The Callout component allows you to highlight important information in your MDX files. It takes an optional emoji prop to add visual interest.

## Basic usage

Here's how to use a basic Callout:

```mdx
<Callout>
  This is a standard callout without an emoji.
</Callout>
```

Which renders as:

<Callout>
  This is a standard callout without an emoji.
</Callout>

## Styling the Callout

The Callout component uses Tailwind CSS for styling. The default styles include:
- A subtle background color
- Border
- Rounded corners
- Appropriate padding
- Emoji positioned to the left of the content

You can customize the appearance by editing the `Callout` component in `src/components/blog/MDXComponents.tsx`.
